## 🔬 About Mortiscope

**Mortiscope** is a thesis project developed by two researchers focused on providing Post-Mortem Interval (PMI) estimation by analyzing the developmental stages of _Chrysomya megacephala_ (oriental latrine blowfly) specimens found at forensic images.

## 🚀 Getting Started

If you wish to run the Mortiscope system locally, please view the setup documentation in the respective repositories:

- [Mortiscope API](https://github.com/mortiscope/mortiscope-api#readme)
- [Mortiscope Web](https://github.com/mortiscope/mortiscope-web#readme)

## 🤝 Contributing
